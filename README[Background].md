# Hypothesis-First-A-B-Testing---Testing-Android-vs-Web-User-Experience
## Background: 
### Client Story:
Xero is a Swedish e‑commerce company that has grown rapidly in recent years. They offr a wide range of consumer products through their online web store. Theere has been an increasing mobile usage among its customer base that led them to recognize the opportunity to deepen engagement and improve the user experience by launching a dedicated Android app on the Google Play Store.

The Android app was launched with a promise  provide users with a more seamless, and mobile‑optimized shopping experience compared to the traditional web interface. The company’s managment believed the Android app could lead to higher user engagement, improved conversion rates, faster/smoother checkout processes, and greater customer satisfaction overall.

To investiagte the actual impact of the Android app relative to the existing web store, Xero decided to conduct a A/B test comparing key usability and performance metrics across both platforms. This analysis will help the company determine whether the Android app delivers measurable improvements in user behavior and satisfaction, and whether it should be prioritized in future development and marketing efforts

### Objectives

The main objectives of this A/B test are to evaluate whether the newly launched Android app provides measurable improvements over the existing Web Store in terms of the following main area:

- User Engagement: to ssess whether users spend more time on the Android app compared to the web store.
- Conversion Rate: to determine if the Android app leads to a higher proportion of users placing orders.
- Checkout Efficiency- to evaluate whether the time from adding items to the cart to completing checkout is shorter on the app.
- Shopping Behavior – to analyze whether users add more items to the cart on the Android app.
- Customer Satisfaction – to measure user satisfaction ratings to determine perceived usability improvements.

### Hypotheses
**Null Hypotheses (H₀):**
H₀₁: There is no difference in average time spent between Android app and Web Store users.

H₀₂: There is no difference in conversion rates between Android app and Web Store users.

H₀₃: There is no difference in time from cart to checkout between platforms.

H₀₄: There is no difference in user satisfaction ratings between platforms.

**Alternative Hypotheses (H₁):**
H₁₁: Users of the Android app spend more time on the platform than Web Store users.

H₁₂: The Android app has a higher conversion rate than the Web Store.

H₁₃: Users checkout faster on the Android app compared to the Web Store.

H₁₄: Users report higher satisfaction ratings on the Android app than on the Web Store.
